### Accessing Surgeons

1.  Click **Admin** in the main navigation
2.  Select **Surgeons**

### Adding a New Surgeon

1.  Click **Add Surgeon**
2.  Enter the surgeon’s details:
    -   Name (required)
    -   Credentials (e.g., MD, FACS)
    -   Specialty
    -   Active status
3.  Click **Save**

### Surgeon-Procedure Assignments

Each surgeon can have specific procedures assigned to them. When creating a quote, only the procedures assigned to the selected surgeon will be available.

To manage assignments:

1.  Go to the Procedures page
2.  Use the surgeon filter to see which procedures are assigned
3.  Toggle procedures on/off for each surgeon
