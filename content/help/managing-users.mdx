### Accessing User Management

1.  Click **Admin** in the main navigation
2.  Select **Users**

### User Roles

-   **Admin** – Full access to all features including settings, user management, and all quotes
-   **User** – Can create and manage quotes but cannot access admin settings

### Adding a New User

1.  Click **Add User**
2.  Enter their details:
    -   Name
    -   Email (used for login)
    -   Role (Admin or User)
3.  Click **Save**
4.  The new user will receive an email with login instructions

### Editing Users

Click on any user to edit their details or change their role.

### Deactivating Users

To remove a user’s access, edit their account and set their status to Inactive. This preserves their quote history while preventing login.
