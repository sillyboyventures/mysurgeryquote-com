### Accessing Procedures

1.  Click **Admin** in the main navigation
2.  Select **Procedures**

### Adding a New Procedure

1.  Click **Add Procedure**
2.  Fill in the details:
    -   Name (required)
    -   Category (e.g., Breast, Body, Face)
    -   Code (optional – for internal reference)
    -   Base Price
    -   Surgeon Time (in minutes)
    -   Notes (optional – appears on quote if enabled)
3.  Click **Save**

### Editing Procedures

Click on any procedure to edit its details. Price changes are tracked in the price history.

### Assigning Procedures to Surgeons

Use the surgeon toggle on the Procedures page to show/hide procedures by surgeon. This controls which procedures appear when creating a quote for each surgeon.

### Procedure Categories

Organize procedures into categories for easier navigation in the quote builder. Common categories include:

-   Breast
-   Body
-   Face
-   Non-Surgical
