### Bulk Import Options

Instead of adding procedures and fees one at a time, you can upload them in bulk using a spreadsheet.

### Preparing Your Spreadsheet

**For Procedures:**

Create a CSV or Excel file with these columns:

| Name | Category | Code | Price | Surgeon Time (min) | Notes |
| --- | --- | --- | --- | --- | --- |
| Breast Augmentation | Breast | BA001 | 5500 | 90 |  |
| Tummy Tuck | Body | TT001 | 8000 | 180 | Includes lipo |

**For Additional Fees:**

| Name | Code | Amount | Description |
| --- | --- | --- | --- |
| Facility Fee | FAC01 | 1500 | Operating room |
| Anesthesia | ANES01 | 800 | General anesthesia |

**For Implants:**

| Brand | Type Name | Code | Price |
| --- | --- | --- | --- |
| Allergan | Smooth Round 350cc | NAT-350 | 1200 |
| Mentor | MemoryGel Moderate 375cc | MEM-375 | 1100 |

### Import Process

1.  Go to **Admin > Procedures** (or Fees/Implants)
2.  Click **Import** or **Upload**
3.  Select your spreadsheet file
4.  Map your columns to the required fields
5.  Preview the import
6.  Click **Confirm Import**

### Tips for Successful Imports

-   **Use consistent formatting** – Keep category names identical (e.g., always “Breast” not sometimes “Breast Procedures”)
-   **Check for duplicates** – The system will warn you if a procedure code already exists
-   **Start small** – Test with 5-10 items first to verify formatting
-   **Keep a backup** – Save your original spreadsheet

### After Importing

Once imported:

-   Review all items for accuracy
-   Assign procedures to surgeons
-   Test by creating a sample quote
