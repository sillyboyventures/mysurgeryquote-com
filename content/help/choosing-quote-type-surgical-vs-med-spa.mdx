For practices offering both surgical and med spa services, you’ll choose the quote type when starting a new quote.

**Starting a New Quote**

1.  Click **New Quote** from the dashboard or sidebar
2.  If your practice type is “Both”, you’ll see a selection screen
3.  Choose:
    -   **Surgical Quote** – For surgical procedures
    -   **Med Spa Quote** – For non-surgical treatments
4.  The quote builder adapts to your selection

**What Changes by Quote Type**

| Feature | Surgical | Med Spa |
| --- | --- | --- |
| Facility Fee | Yes | No |
| Anesthesia Fee | Yes | No |
| Implants | Yes | No |
| Unit-Based Pricing | No | Yes |
| Membership Tiers | No | Yes |
| Treatment Series | No | Yes |
| Time Calculations | Yes | No |

**Changing Quote Type**

Quote type cannot be changed after creation. If you selected the wrong type, create a new quote with the correct type.
