### Finding Quotes Quickly

The Quotes page provides powerful search and filter options to find any quote instantly.

### Search by Patient

Type in the search box to find quotes by:

-   Patient first name
-   Patient last name
-   Quote number

Results update as you type.

### Filter by Status

Click the status filter to show only:

-   **All** – Every quote
-   **Draft** – Unsent quotes
-   **Sent** – Emailed to patient
-   **Accepted** – Patient accepted
-   **Expired** – Past validity date

### Filter by Date

Use the date range picker to find quotes:

-   Created today
-   This week
-   This month
-   Custom date range

### Filter by Surgeon

If your practice has multiple surgeons, filter to see quotes for a specific surgeon only.

### Combining Filters

All filters work together. For example:

-   Show all “Sent” quotes from “Dr. Smith” in “December 2024”

### Sorting Results

Click column headers to sort by:

-   Date (newest/oldest)
-   Patient name (A-Z, Z-A)
-   Total amount (high/low)
-   Status
