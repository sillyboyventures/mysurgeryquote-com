Procedure notes let you add custom text that appears on the patient quote, indented below the procedure name.

**Enabling Notes for a Procedure**

1.  Go to **Admin → Procedures**
2.  Edit the procedure
3.  Check **“Has Notes Field”**
4.  Save

**Common Uses**

-   Treatment areas (e.g., “Liposuction – Flanks and Abdomen”)
-   Specifications (e.g., “50 units – Forehead, Glabella, Crow’s Feet”)
-   Inclusions (e.g., “Includes compression garment”)
-   Custom details for the patient

**Adding Notes During Quote Creation**

1.  Add a procedure that has notes enabled
2.  A text field appears below the procedure
3.  Enter your note
4.  The note appears on the PDF indented below the procedure name

**On the Patient Quote**

```
Liposuction                                    $4,500
    Flanks and Abdomen
```
