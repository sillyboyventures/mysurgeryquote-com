### Accessing Implants

1.  Click **Admin** in the main navigation
2.  Select **Implants**

### Understanding the Implant Structure

Implants are organized in two levels:

1.  **Brands** – The manufacturer (e.g., Allergan, Mentor, Sientra)
2.  **Implant Types** – Specific products under each brand (e.g., sizes, styles, profiles)

### Adding a New Brand

1.  Click **Add Brand**
2.  Enter the brand name
3.  Click **Save**

### Adding Implant Types

1.  Select a brand
2.  Click **Add Implant Type**
3.  Enter the details:
    -   Name (e.g., “Smooth Round Moderate Profile 350cc”)
    -   Code (optional – manufacturer reference)
    -   Price
4.  Click **Save**

### Editing Implants

Click on any brand or implant type to edit. Price changes are tracked in the price history for audit purposes.

### Using Implants in Quotes

When creating a quote:

1.  Select the implant brand
2.  Choose the specific implant type
3.  Enter quantity
4.  Apply discounts if applicable (e.g., manufacturer promotions)
