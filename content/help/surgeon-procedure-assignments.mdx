Control which surgeons can perform which procedures. This filters the procedure list based on the selected surgeon.

**Setting Up Assignments**

1.  Go to **Admin → Procedures**
2.  Click the **Surgeon Toggle** icon on a procedure
3.  Check the surgeons who perform this procedure
4.  Save

**Or from the Surgeon side:**

1.  Go to **Admin → Surgeons**
2.  Edit a surgeon
3.  Go to **Procedure Assignments**
4.  Check the procedures they perform
5.  Save

**In the Quote Builder**

When a surgeon is selected, only their assigned procedures appear in the procedure list. This prevents accidentally adding procedures the surgeon doesn’t perform.

**Unassigned Procedures**

If a procedure has no surgeon assignments, it appears for all surgeons (backward compatible behavior).
