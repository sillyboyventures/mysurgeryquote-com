This guide walks you through connecting your ModMed account to MySurgeryQuote.

### Before You Begin

You’ll need to request API access from ModMed. This is a one-time setup handled by their team.

### Step 1: Request API Integration

Email **[practicesolutions@modmed.com](mailto:practicesolutions@modmed.com)** with the following:

> Subject: API Integration Request – MySurgeryQuote
> 
> Hello,
> 
> We would like to enable API integration between our ModMed account and MySurgeryQuote (Patient Quote Builder).
> 
> Practice Name: \[Your Practice Name\]  
> Firm URL Prefix: \[Your ModMed URL prefix, e.g., “yourpractice” from yourpractice.ema-api.com\]
> 
> Please provide us with the API credentials needed for this integration.
> 
> Thank you.

ModMed’s Practice Solutions team typically responds within 2-3 business days.

### Step 2: Enter Your Credentials

Once you receive your credentials from ModMed:

1.  Go to **Admin → EMR Integration**
2.  In the ModMed section, enter:
    -   **Firm URL Prefix** – Your practice identifier (e.g., “yourpractice”)
    -   **Username** – API username provided by ModMed
    -   **Password** – API password provided by ModMed
    -   **API Key** – API key provided by ModMed
3.  Click **Save Credentials**

### Step 3: Test the Connection

1.  Click **Test Connection**
2.  You should see a green “Connected” status
3.  If the test fails, double-check your credentials

### Step 4: Enable the Integration

1.  Toggle **Enable ModMed Integration** to ON
2.  The integration is now active

### Sending Quotes to ModMed

Once connected:

1.  Open any quote
2.  Click **Send to ModMed**
3.  Search for the patient by name
4.  Select the patient from the results
5.  Choose what to send:
    -   **Quote PDF** – Attaches the PDF to patient documents
    -   **Billing Charges** – Creates charges in ModMed (optional)
6.  Click **Send**
