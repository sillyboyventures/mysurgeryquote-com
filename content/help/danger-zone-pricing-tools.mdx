The Danger Zone contains powerful tools for managing pricing across your entire practice. Use with caution.

**Accessing Danger Zone**

1.  Go to **Admin → Settings**
2.  Scroll to **Danger Zone** (Admin or higher required)

**Available Tools**

**Export All Pricing**  
Downloads an Excel file with all procedures, surgeon overrides, and current pricing. Useful for review or backup.

**Save Current Prices as Baseline**  
Saves a snapshot of all current prices. This becomes your “original” price that can be restored later. Use before making bulk changes.

**Restore Original Prices**  
Reverts all procedure prices to the saved baseline. Useful if you need to undo bulk changes.

**Delete All Pricing**  
Removes all procedure pricing data. Requires multiple confirmation steps. This is destructive and cannot be undone.

**Best Practices**

1.  Always **Save Baseline** before bulk changes
2.  **Export Pricing** as a backup before major changes
3.  Only use **Delete All** when completely rebuilding your price list
